# NaoTodo

#### 介绍

Nao待办平台 - 前端项目  
Nao待办平台是一个基于 Vue3 + Vite + NueUI + Pinia + Axios + TypeScript 的待办事项管理系统。  
Nao待办平台的目标是打造一个简单、易用、功能丰富的待办事项管理系统，让用户可以快速、高效地管理自己的工作任务。

#### 软件架构

1. 使用 Vue3 作为前端框架
2. 使用 Vite 作为开发环境
3. 使用 NueUI 作为 UI 组件库
4. 使用 Pinia 作为状态管理库
5. 使用 Axios 作为 HTTP 库
6. 使用 TypeScript 作为开发语言
7. 使用 ESLint 作为代码检查工具
